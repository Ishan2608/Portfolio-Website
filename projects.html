Text To Morse Code

This program has the ability to encrypt english. text to morse code as well as decrypt morse code to english text. The program has been implemented using python's one of in-built data structure, i.e., dictionaries. The english letters are the keys and their equivalent code is the value of that key.

Tic Tac Toe

The classic tic tac toe game that you can play on console. This game utilized python lists, here 2D lists that keep the matrix table of tic tac toe. Value insertion is done through direct accessing and winner is checked through traversal of rows, columns and diagonals, counting the number of O and X.
